# resources/ 目录说明

## icon.ico
应用图标（当前为占位图标）
替换方法：将你的 256x256 ICO 文件命名为 icon.ico 放在此目录

推荐尺寸：256x256 像素，包含 16/32/48/256 多尺寸的 ICO 文件
可用工具：https://www.icoconverter.com/ 将 PNG 转为 ICO

## arduino-cli.exe（可选）
将从以下地址下载的 arduino-cli.exe 放到此目录，可内置到安装包中：
https://github.com/arduino/arduino-cli/releases/latest

下载：arduino-cli_0.xx.x_Windows_64bit.zip → 解压出 arduino-cli.exe → 放到此目录

如果不放此文件，用户需要自己安装 arduino-cli 并加入 PATH。
